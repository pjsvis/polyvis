# browser allow list

The browserAllowList.txt file contains a fairly permissive list of domains that are allowed to be opened in the browser.

We have switched browser permissions to `turbo` mode which allows the browser-agent to use `getComputedStyle` to inspect CSS computed styles.

We should keep this under review and update the list as needed.
